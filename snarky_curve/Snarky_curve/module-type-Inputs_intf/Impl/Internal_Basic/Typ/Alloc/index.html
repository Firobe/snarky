Module Typ.Alloc

A 'value Alloc.t describes allocating variables for the R1CS to use without explicitly giving the values to store in them. This is analogous to Store.t.

The main use of this is in generating the constraint system and generating the Keypair.t with generate_keypair; we can't know yet what values we will want to store in the variables, but we still want to know what constraints they will have to satisfy.
